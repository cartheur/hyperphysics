Doppler Effect Wavelength Calculation

At temperature C = F

the sound speed in air is m/s.

If the source frequency is Hz, then the source wavelength is m.

If the velocity of the source is m/s = mi/hr

then for an approaching source the wavelength is m

and for a receding source the wavelength is m.

Note: The frequency will default to A4 (440 Hz) and the temperature will default to 20 C if those values are not entered. Any parameters can be changed.
