System Matrix, Hecht Form

For systems of multiple thick lenses, it is sometimes useful to represent the system by a system matrix. The matrix is built up by multiplying the refraction matrices and translation matrices.

This is the form of the system matrix used by Hecht. An alternate form is used by Meyer-Arendt.
