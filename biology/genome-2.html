Genome

Genome is a general term which refers to the set of genetic information of a cell, including the information for constructing proteins, RNA, and other cellular components. The genetic information is stored in the DNA of the cell using the genetic code. The genome contains the blueprints for the construction of the essential proteins, residing in functional segments called genes. In addition to the protein blueprints, the genome contains information which provides for expression and regulation of the genes.

From an information theory perspective, the genome is a unique set of programs embedded in the DNA memory for an organism.
